## Overview

Generating new from CNN, Fox News, NY Times, and Ruters
This application aggregates and presents news content, providing a streamlined way to consume information from multiple sources.

The project contains **3** files across **3** languages: HTML, JavaScript, Markdown.

## Features

- 📰 Aggregates news from multiple sources
- 🔄 Real-time content updates
- 🏷️ Category-based news filtering
- 📱 Responsive design for all devices
- 🔍 Search functionality

## Getting Started

### Prerequisites

- A modern web browser

### Installation

1. **Clone the repository**

   ```bash

   cd News-Generator
   ```


## Usage

Open `index.html` in your web browser to run the application.

```bash
# Or use a local server
npx serve .
```
